# 🎲 Betis - Arbitrage Scanner

Scanner de apuestas deportivas que busca arbitrajes automáticamente en tiempo real.

## Características
- ✅ Betfair API (tiempo real, legal, gratis)
- ✅ 1xBet scraping (sin detección)
- ✅ 888sport scraping (sin detección)
- ✅ Busca arbitrajes cada minuto
- ✅ Múltiples deportes
- ✅ Calcula stakes automáticamente
- ✅ Retorno asegurado garantizado

## Requisitos
- Python 3.8+
- Cuenta Betfair (gratis)
- API Key Betfair (gratis)

## Instalación Rápida

### 1. Clonar repositorio
```bash
git clone https://github.com/TU_USUARIO/betis-arbitrage.git
cd betis-arbitrage
```

### 2. Crear entorno virtual
```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Instalar dependencias
```bash
pip install -r requirements.txt
```

### 4. Configurar credenciales
```bash
copy .env.example .env
# Edita .env con tus credenciales
```

### 5. Ejecutar
```bash
python scanner_hibrido.py
```
